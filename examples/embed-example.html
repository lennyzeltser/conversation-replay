How It Works

When the player detects it's running inside an iframe, it automatically adjusts its styling for seamless embedding. The background becomes transparent, padding is removed, and the player blends naturally with the parent page.

The tab navigation now supports horizontal scrolling with clear visual indicators. Double-chevron arrows appear when there are more tabs to reveal, making it easy for users to discover all available scenarios.
